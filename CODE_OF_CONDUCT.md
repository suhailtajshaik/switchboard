# Code of Conduct

This project adopts the **Contributor Covenant v2.1**
(https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, welcoming, and constructive; harassment and personal
attacks are not tolerated. Report issues privately to the maintainer contact
listed in `SECURITY.md`. Maintainers may edit, remove, or reject contributions
and contributors that violate this code.
